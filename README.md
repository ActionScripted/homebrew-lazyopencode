# homebrew-lazyopencode

Homebrew tap for [lazyopencode](https://github.com/actionscripted/lazyopencode) — a terminal UI for managing [opencode](https://opencode.ai) sessions.

## Install

```sh
brew install actionscripted/lazyopencode/lazyopencode
```

Or tap first, then install:

```sh
brew tap actionscripted/lazyopencode
brew install lazyopencode
```

## Upgrade

```sh
brew upgrade lazyopencode
```
